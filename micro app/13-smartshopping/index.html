⚖️ Privacy & Condizioni d'Uso

Informativa sulla Privacy & GDPR (Suite Offline Locale)

1. Trattamento dei Dati Locali (100% Client-Side)
Questa applicazione e tutte le sue micro-app collegate sono progettate per essere eseguite interamente nel browser dell'utente. Nessun dato inserito (liste della spesa, scadenze, idee personali, transazioni finanziarie, password o dati di configurazione) viene inviato, salvato o trasmesso a server esterni o terze parti.

2. Uso del LocalStorage
Tutti i dati vengono memorizzati localmente sul dispositivo dell'utente tramite la tecnologia localStorage del browser. L'utente ha il controllo totale su questi dati e può eliminarli in qualsiasi momento svuotando la cache del browser o utilizzando le impostazioni di ripristino/elimina integrate.

3. Notifiche Locali
Le notifiche push/avvisi del timer utilizzano le Web Notification API native del browser dell'utente. Vengono generate localmente sul dispositivo e non richiedono alcuna comunicazione con server remoti. I permessi per le notifiche possono essere revocati in qualsiasi momento tramite le impostazioni del browser.

4. Cookie e Tracciamento
La suite non utilizza cookie di tracciamento, codice analitico di terze parti (come Google Analytics o Facebook Pixel), né pubblicità mirata che raccoglie dati personali.

5. Limitazione di Responsabilità
L'applicazione è fornita "così com'è" senza garanzie di alcun tipo. L'autore non è responsabile per la perdita accidentale di dati causata dalla cancellazione spontanea della memoria del browser o da anomalie del dispositivo.
